### `delete_asset` <sup><a href="https://github.com/PrefectHQ/prefect/blob/main/src/prefect/cli/cloud/asset.py#L106" target="_blank"><Icon icon="github" style="width: 14px; height: 14px;" /></a></sup>

The key should be the full asset URI (e.g., 's3://bucket/data.csv').
