The [Prefect REST API](/3.0rc/api-ref/rest-api/) is used to send and receive data for orchestration and monitoring between Prefect clients and Prefect Cloud or a self-hosted Prefect server instance.
This API is mainly consumed by Prefect clients such as the Prefect Python Client or the Prefect UI.
